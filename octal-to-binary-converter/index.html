Octal to Binary Conversion

The octal to binary conversion takes place in four different parts. The first part is obviously separating each octal number. Once you separate the octal numbers, find the equivalent binary numbers for those octal numbers. Do note that if you find any binary number shorter then 3 bits then do add zeros in the particular number. Write down all the binary number together and you will get the equivalent binary number to that octal number.
